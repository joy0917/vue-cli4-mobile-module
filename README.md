# vue-cli4-mobile-module
基于vue-cli4 router axios vuex vant-ui less配置的移动端项目模板（自用）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
